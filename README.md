# 🥖 Baguette 

Exemple de jeu de combat au tour par tour sous Unity 5.6.6f2

Aucun package requis et Utilise C# 4.0

## 🎮 Fonctionnalités

    -⚔️ Combat au tour par tour
    -Exploration en 3D isométrique (actuellement jouable uniquement via manette)
